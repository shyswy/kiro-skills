# java-spring

> Java + Spring Boot 개발 패턴 가이드 — Java 17/21, Spring Boot 3.x, JPA, Gradle/Maven

## When to Use

- Java, Spring Boot, Spring, Maven, Gradle
- JPA, Hibernate, Spring Security, Virtual Threads
- 자바, 스프링, 스프링부트, 메이븐, 그래들, 백엔드, 서버

- 프로젝트 구조 (Layered / Multi-Module)
- Java 17+ 패턴 (Record, Sealed Class, Pattern Matching, Virtual Threads)
- Spring Boot 설정 (Gradle Kotlin DSL, Maven, application.yml)
- Controller / Service / Repository 패턴
- 글로벌 에러 핸들링 (@RestControllerAdvice)
- 테스트 (JUnit 5 + Mockito, Slice Test, Integration Test)
- 성능 및 운영 (N+1, 캐시, 모니터링)
