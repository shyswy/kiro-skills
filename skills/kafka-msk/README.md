# kafka-msk

> Apache Kafka / AWS MSK 메시징 패턴 가이드 — Consumer/Producer 설계, Kafka Streams, KTable, exactly-once

## When to Use

- Kafka, MSK, consumer, producer, topic, partition
- Kafka Streams, KTable
- 메시징, 이벤트 스트리밍, 메시지 큐, 스트림 처리

## What It Covers

- Consumer/Producer 설계 패턴
- Kafka Streams & KTable (stateful processing)
- Exactly-once semantics (EOS)
- Partition 전략 및 consumer group 관리
- AWS MSK 특화 설정 (IAM auth, MSK Connect, schema registry)
- Dead Letter Queue (DLQ) 패턴
- Offset 관리 전략
