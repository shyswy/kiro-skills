# typescript-node

> TypeScript/Node.js 개발 패턴 가이드 — NestJS, Express, Fastify, 모듈 설계, 비동기 처리

## When to Use

- TypeScript, Node.js, NestJS, Express, Fastify, npm
- 모듈 설계, 비동기, DI, 의존성 주입
- 타입스크립트, 노드, 백엔드, 서버, 에러 핸들링

## What It Covers

- 프로젝트 구조 (Layered modules 패턴)
- NestJS 패턴 (Module, Provider, Controller, Guard, Pipe)
- Express/Fastify 미들웨어 패턴
- 의존성 주입 (DI) 설계
- 비동기 처리 (Promise, async/await, 동시성 제어)
- 에러 핸들링 (Global filter, custom exceptions)
- 테스트 전략 (unit, integration, e2e)
- tsconfig 최적화 및 빌드 설정
