# kotlin-spring

> Kotlin + Spring Boot 개발 패턴 가이드 — Spring Boot 3.x, WebFlux, Coroutines, JPA, 테스트

## When to Use

- Kotlin, Spring Boot, Spring, WebFlux, 코루틴
- Gradle Kotlin DSL, Spring Data JPA, R2DBC
- 코틀린, 스프링, 스프링부트, 그래들, 백엔드, 서버

## What It Covers

- 프로젝트 구조 (Layered / Hexagonal)
- Kotlin 핵심 패턴 (data class, sealed class, extension, value class)
- Spring Boot 설정 (Gradle Kotlin DSL, application.yml)
- Controller / Service / Repository 패턴
- 글로벌 에러 핸들링 (@RestControllerAdvice)
- 비동기 처리 (Coroutines + WebFlux)
- 테스트 (JUnit 5 + MockK, Slice Test, Integration Test)
- 성능 및 운영 (N+1, 캐시, 모니터링)
