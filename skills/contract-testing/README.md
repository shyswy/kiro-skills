# contract-testing

> Producer-Consumer 이벤트 계약 테스트 패턴

## When to Use
- 계약 테스트 작성/설계 시
- Producer fixture 정의 시
- Consumer schema 변경 시 호환성 확인
- Event-Driven 서비스 간 통신 계약 관리

## What It Covers
- Producer Fixture 정의 패턴
- Consumer Schema (Joi/Zod) 교차 검증
- 계약 깨짐 감지 및 대응 프로세스
- CI gate 연동
- 다중 레포 환경 계약 관리
